# Form Registrasi Android

Aplikasi form registrasi Android menggunakan Kotlin dan Material Components.

## Fitur
- Input data pribadi (nama, email, password)
- Validasi real-time
- RadioButton jenis kelamin
- Checkbox hobi (minimal 3)
- Spinner kota asal
- AlertDialog konfirmasi
- Long press gesture

## Teknologi
- Kotlin
- Android Studio
- Material Components
- AppCompat

## Cara Menjalankan
1. Clone repository ini
2. Buka dengan Android Studio
3. Run ke device/emulator
